## Перед началом

### Цель

Добавить команду `/help`, доступную только создателю бота, и показать в ней полный актуальный перечень всех зарегистрированных slash-команд Velvet вместе с назначением и совместимыми алиасами.

### Исходный контекст

В BotFather-меню владельца были оставлены только компактные команды `/start`, `/menu` и `/archive`, тогда как код поддерживал около девяноста служебных маршрутов. Существующий integrity-тест учитывал декораторы `Command`, но не обеспечивал отдельный пользовательский справочник и не проверял программные регистрации через `router.message.register`.

### Планируемый объём

- добавить отдельный owner help controller;
- перечислить все команды по функциональным разделам;
- разбить длинный ответ на сообщения, укладывающиеся в лимит Telegram;
- добавить `/help` только в scoped-меню владельца;
- не добавлять `/help` публичным пользователям и модераторам;
- подключить новый роутер к core operations;
- добавить тест фактического покрытия зарегистрированных команд.

### Критерии готовности

- `/help` обрабатывается ботом и возвращает полный справочник;
- справочник содержит `CommandStart`, декораторные команды и команды из `message.register`;
- каждая страница короче 4096 символов;
- команда отображается только в owner command menu;
- публичное и moderator command menu не меняются;
- тесты проверяют совпадение справочника с реальными маршрутами.

### Риски и ограничения

Доступ опирается на существующий `OwnerAccessMiddleware` и owner allowlist. Справочник остаётся статическим представлением назначения команд, поэтому тест контролирует состав команд, но текстовые описания при изменении поведения всё равно требуют осознанного обновления.

## После завершения

### Фактически сделано

- создан `owner_help.py` с 91 уникальной командой и алиасом, сгруппированными по девяти разделам;
- ответ автоматически разбивается на две страницы;
- `/help` добавлен в `build_admin_commands`, но отсутствует в public и editor menus;
- owner help router подключён к `core_operations`;
- добавлен AST-тест, который находит `CommandStart`, декораторы `Command` и программные `message.register`;

### Миграции и совместимость

SQL-миграции не требуются. Существующие команды, callback contracts и роли не изменены. `/help` использует действующую матрицу доступа: владельцы проходят по allowlist, модераторы и публичные пользователи команду не получают.
