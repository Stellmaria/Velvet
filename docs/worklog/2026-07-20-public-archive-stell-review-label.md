## Перед началом

### Цель

Заменить технический owner ID в административной подписи публичного архива на понятное имя «Стэл», не меняя логику определения просмотра владельцем.

### Исходный контекст

Карточка менеджера показывала строку `Просмотр 7221553045`, хотя числовой ID нужен только внутренней логике и базе данных. Пользователь просит отображать человеческое имя.

### Планируемый объём

- изменить только текст административной подписи;
- оставить `reviewed_by_owner` и owner ID без изменений;
- обновить regression-тест;
- не затрагивать watermark, скачивание и отложенный channel analytics.

### Критерии готовности

- карточка содержит `Просмотрено Стэл`;
- карточка больше не выводит owner ID в этой строке;
- существующие статусы `просмотрено / не просмотрено` сохраняются;
- CI проходит.

### Риски и ограничения

Изменение только presentation-уровня. Данные просмотров, пользовательские ID и правила доступа не мигрируются.

## После завершения
