# Канонические требования проекта Velvet

Файлы в этом каталоге имеют статус проектного ТЗ. Реализация, миграции, Telegram-интерфейсы и worklog должны ссылаться на соответствующий документ.

## Активные ТЗ

- [`workspace_product.md`](workspace_product.md) — пользовательские пространства, разрешения, модули, приватность, workspace taxonomy, публичные архивы и роли.

При расхождении старого worklog или устаревшего комментария с ТЗ приоритет имеет этот каталог, если владелец проекта явно не утвердил изменение требований.
